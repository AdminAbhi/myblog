# This is a simple blog Website

## Features
  1. Responsive UI
  2. Navigation Bar
  3. Add blog
  4. Edit blog
  5. Delete Blog
  
## Technology Stack:
  1. HTML
  2. CSS
  3. Bootstrap 4
  4. JavaScript
  5. JQuery
  6. NodeJS
  7. MongoDB
